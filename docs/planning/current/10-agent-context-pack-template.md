# 10. Agent Context Pack Template

Before a coding agent receives work, provide only the following:

1. `planning-pack.yml`
2. `README.md`
3. the specific slice pack marked READY_FOR_CODING
4. referenced resource cards from `05-resource-cards.md`
5. referenced action cards from `06-action-cards.md`
6. relevant state transitions from `13-state-transition-map.yml`
7. relevant tests from `14-test-matrix.yml`
8. locked decisions from `11-open-decisions.md`

Agents SHALL NOT infer behaviour from broad project vision, old Store_Blueprint files, or adjacent deferred domains.
